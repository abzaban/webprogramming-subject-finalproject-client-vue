# webprogramming-subject-finalproject-client-vue
This showcases the frontend of my final project, built with Vue.js as part of the web programming course.

# Tech Stack
<ul>
  <li>Node</li>
  <li>NPM</li>
  <li>Vue</li>
  <li>JavaScript</li>
</ul>

# Run project
Open a CLI in root folder of project and then run:

```sh
npm install
npm run serve
```
